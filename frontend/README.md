## DocChat Frontend

A modern React + Vite single-page app that provides a polished UI for the DocChat backend.

### Development

- **Install dependencies**

```bash
cd frontend
npm install
```

- **Run the dev server**

```bash
npm run dev
```

By default the app expects the backend FastAPI server to be running at `http://localhost:8000`.
You can override this by creating a `.env` file and setting:

```bash
VITE_API_BASE_URL=http://localhost:8000
```
